Detailed Description

template<typename T, uInt N, uInt d>
class EDMatrix< T, N, d >

EDMatrix This file contains the implementations of the template class EDMatrix and some external methods usefull to work with Euclidean Distance Matrix

Requirments: Eigen 3.3.7 I've performed test by using Catch v2.11.0

to build and works with Euclidean

Distance Matrix(EDM)

Template Parameters
Ttype of entries of the EDM, can be double, float.
Nnumber of points from which EDM is created. Is the dimension of the NxN EDM.
ddimension of the euclidean space in which points live
