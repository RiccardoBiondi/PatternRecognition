Detailed Description

template<typename T, unsigned int N, unsigned int d>
class Procrustes< T, N, d >

EDMatrix v2.0.0

Starting Generation: 2020-03-20

This file contains the implementations of the template class Prcrustes that contains tools to perform the Procrustes alignment of two set of *point in the euclidean space

Requirments: Eigen 3.3.7, in particular the module Eigenvalues and the module SVD I've performed test by using Catch v2.11.0

to perform the prcrustes analysis

Procrustes use Eigen 3.3.7 to work.
